# rx5808_pro_osd
Deluxe OSD 5.8GHz (rx5808 module) control based on minimOSD Ardunino board

This project is a Arduino controller for the Boscam RX5808 FPV receiver.

It is based on the MinimOSD... and has an OSD interface.
